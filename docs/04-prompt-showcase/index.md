# 4장. 프롬프트 쇼케이스

## remotion.dev/prompts

Remotion 팀은 실제 프롬프트로 생성된 영상 예제를 공개적으로 공유하는 쇼케이스 사이트를 운영합니다.

**[https://www.remotion.dev/prompts](https://www.remotion.dev/prompts)**

이 사이트는 다음과 같은 정보를 제공합니다.

- 실제로 사용된 프롬프트 텍스트
- 그 프롬프트가 생성한 영상 미리보기
- 다운로드 가능한 TypeScript 소스 코드
- 컴포지션 설정 (크기, fps, 길이)

---

## 왜 쇼케이스를 먼저 확인해야 하는가

처음부터 프롬프트를 작성하는 것보다, 이미 검증된 예제에서 시작하는 것이 훨씬 효율적입니다.

**초보자에게 특히 유용한 이유:**

1. **가능성의 범위 파악**: Remotion+Codex로 어떤 종류의 영상이 만들어지는지 먼저 확인할 수 있습니다
2. **프롬프트 작성 패턴 학습**: 전문가들이 프롬프트를 어떻게 구성하는지 직접 볼 수 있습니다
3. **코드 시작점 확보**: 처음부터 작성하지 않고, 가장 가까운 예제 코드에서 시작할 수 있습니다
4. **실현 가능성 검증**: 내가 원하는 효과가 이미 만들어진 적 있는지 확인할 수 있습니다
