# 3장. Codex로 모션 그래픽 만들기

## 이 챕터의 핵심

이 챕터는 이 블로그에서 가장 중요한 부분입니다. Codex 프롬프트를 사용해 텍스트 애니메이션, 도형 애니메이션, 씬 전환, 이미지·로고 삽입까지 실제 모션 그래픽의 주요 요소를 모두 만들어봅니다.

코드를 처음부터 작성할 필요 없습니다. 각 페이지에서 제공하는 Codex 프롬프트를 복사해 사용하면 됩니다.

---

## 이 챕터의 구성

| 페이지 | 내용 | 핵심 기술 |
|--------|------|-----------|
| [1. 프롬프트 작성법](./prompt-basics) | 좋은 프롬프트 vs 나쁜 프롬프트 | 프롬프트 구조화 |
| [2. 텍스트 애니메이션](./text-animation) | 페이드인, 슬라이드업, 타이프라이터 | `interpolate`, `spring` |
| [3. 도형 애니메이션](./shape-animation) | 그라디언트, 원형, 다중 요소 | `interpolateColors`, `@remotion/shapes` |
| [4. 씬 전환](./transitions) | fade, slide, wipe 트랜지션 | `@remotion/transitions` |
| [5. 에셋 사용](./assets) | 이미지, 로고 삽입 | `staticFile()` |

---

## 이 챕터를 마치면

- Codex 프롬프트로 모션 그래픽 씬을 생성할 수 있습니다
- 텍스트와 도형에 다양한 애니메이션을 적용할 수 있습니다
- 여러 씬을 전환 효과와 함께 연결할 수 있습니다
- 이미지와 로고를 영상에 삽입할 수 있습니다

4장에서는 Remotion 공식 프롬프트 쇼케이스에서 실제 예제를 보고 커스터마이징하는 방법을 다룹니다.
