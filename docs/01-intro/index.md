# How2Remotion — 시작 페이지

> AI 에이전트로 영상을 만드는 가장 실용적인 방법

이 블로그는 **OpenAI Codex**를 이용해 [Remotion](https://remotion.dev)으로 모션 그래픽, AI 더빙, 영상 편집을 하는 방법을 단계별로 설명합니다.

코드를 처음부터 짤 필요 없습니다. 프롬프트로 지시하면 Codex가 Remotion 코드를 생성하고, 여러분은 결과물을 확인하고 방향만 잡으면 됩니다.

## 이 블로그가 다루는 것

```
프롬프트 입력
    ↓
Codex가 TypeScript Remotion 코드 생성
    ↓
ElevenLabs가 AI 목소리 생성
    ↓
Remotion이 MP4로 렌더링
    ↓
완성된 영상
```

## 이 블로그가 다루지 않는 것

- Remotion API 전체를 외우는 것
- 복잡한 React 아키텍처 설계
- After Effects / Premiere 대체
